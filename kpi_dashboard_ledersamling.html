Hvordan vedlikeholde KPI-data

Metode 1: Rediger direkte i GitHub (anbefalt)

  1. Gå til kpis.json i ditt repo
  2. Klikk på blyant-ikonet (edit)
  3. Oppdater verdiene (current, target, comments osv.)
  4. Klikk "Commit changes"
  5. Dashbordet oppdateres automatisk!

Metode 2: Fra Excel til JSON

Hvis du foretrekker å jobbe i Excel:

  1. Opprett en Excel-fil med kolonnene: title, metric, target, current, comments
  2. Bruk en JSON-konverter: https://www.convertcsv.com/csv-to-json.htm
  3. Eksporter Excel som CSV → konverter til JSON
  4. Kopier JSON-en inn i kpis.json på GitHub

📋 JSON-format (reference)

[
  {
    "id": 1,
    "title": "KPI-navn",
    "metric": "Måleenhet",
    "target": 95,
    "current": 92,
    "comments": "Kommentarer"
  }
]
